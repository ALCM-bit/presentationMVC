# Apresentação com Projeto MVC

## Descrição

Esse projeto foi desenvolvido como um desafio para ser entregue para a empresa Auvo Tecnologia. O desafio consistiu em criar um Projeto ASP.NET.MVC com duas Views staticas:

- Home : Contendo o nome do candidato; Cargo pretendido; Pretensão salarial; Data do teste.
- About : Com um texto informando qual desafio de programação mais marcou sua jornada até o momento. Neste deve-se colocar: Tecnologia usada, Arquitetura, Como você atuou e por que te marcou tanto este desafio.



## Tecnologias Utilizadas
- Linguagem C#
- IDE Visual Studio
- POO (Programação Orientada a Objetos)
- ASP.NET MVC
- HTML
- CSS
